# 🧪 מחשבון קצב גדילה מודולרי (Python, SciPy, Matplotlib)

זהו פרויקט **Python מודולרי** לחישוב וניתוח **קצב הגדילה הספציפי ($k$)** של אוכלוסיות ביולוגיות (למשל חיידקים או תאים) באמצעות **נתוני זמן-ריכוז (Time-Series)**.

## ✨ תכונות עיקריות

* **חישוב נקודה בודדת ($k$)**: מבוסס על $N_0$ ו-$N_t$.
* **התאמה מרובת נקודות**: שימוש ב**רגרסיה ליניארית** (`scipy.stats.linregress`) לניתוח מדויק של עקומת הגדילה.
* **ממשק גרפי (GUI)**: טעינת נתונים, שרטוט גרפים (`matplotlib`), הצגת תוצאות.
* **טעינת קבצים**: אפשרות לטעון נתונים ישירות מקובץ CSV או TXT.

---

## 📈 עקרון הרגרסיה הליניארית

צמיחת תאים היא **מעריכית**, אך רגרסיה ליניארית דורשת נתונים ליניאריים. לכן, מבצעים טרנספורמציה לוגריתמית בסיס 2:

[
\log_2(N) = \log_2(N_0) + k \cdot t
]

* השיפוע של הקו המתאים = **קצב הגדילה $k$**
* $R^2$ מציין את מידת ההתאמה של הקו לנתונים (1 = התאמה מושלמת)

> ⚠️ **הערת שימוש:** הזינו רק נתונים משלב **הגדילה המעריכית** לקבלת $k$ מדויק. ניתן להשתמש ב-$R^2$ לבחירת הנקודות הנכונות.

## 🖼️ דוגמאות גרפים

לאחר החישוב, ה-GUI מציג גרף **log2(ריכוז) מול זמן** עם קו הרגרסיה הליניארי:

```python
import matplotlib.pyplot as plt
import numpy as np
from scipy.stats import linregress

t = np.array([0, 1.5, 3.0, 4.5, 6.0])
N = np.array([0.1, 0.25, 0.5, 1.05, 2.0])
logN = np.log2(N)

slope, intercept, r_value, _, _ = linregress(t, logN)

plt.scatter(t, logN, color='blue', label='נקודות נתונים')
plt.plot(t, slope*t + intercept, color='red', label=f'קו התאמה: k={slope:.2f}, R²={r_value**2:.2f}')
plt.xlabel('זמן (שעות)')
plt.ylabel('log2(ריכוז)')
plt.legend()
plt.show()
```
